# Gestion-eventos

API RESTful para la gestión de eventos, desarrollada con Node.js, Express y Sequelize. Aplica el patrón de arquitectura MVC y permite administrar eventos y sus categorías mediante operaciones CRUD completas.

# Tecnologías utilizadas


Node.js — Entorno de ejecución
Express — Framework para el servidor HTTP
Sequelize — ORM para la gestión de la base de datos
MySQL — Motor de base de datos
dotenv — Manejo de variables de entorno
